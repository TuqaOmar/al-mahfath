# 📖 محفظ AI | Ma7fath AI

> **منصة تفاعلية ذكية لمساعدة حفاظ القرآن الكريم على تخطيط وتثبيت ومراجعة حفظهم باستخدام الذكاء الاصطناعي.**  
> *An AI-powered interactive platform to help Quran memorizers plan, review, and track their progress using Google Gemini AI.*

---

## 🌍 نظرة عامة | Overview

**محفظ AI** هو تطبيق ويب متكامل ومصمم بشكل عصري لمرافقة حافظ القرآن الكريم في رحلته الإيمانية. يجمع التطبيق بين قوة الذكاء الاصطناعي (ممثلاً في نموذج **Google Gemini**) والأساليب العلمية والعملية لتثبيت الحفظ، مثل التكرار المتباعد، التقييم اللوني لصفحات المصحف، وتحديد أنماط التعلم المفضلة.

**Ma7fath AI** is a modern web application designed to accompany Quran memorizers on their journey. It combines the power of Google Gemini AI with practical memorization systems such as spaced repetition, interactive color-coded tracking for the 604 pages of the Quran, and personalized learning style profiling.

---

## ✨ الميزات الرئيسية | Key Features

### 1. الخريطة التفاعلية للمصحف (604 صفحات) 🗺️
* تتبع بصري ملون لكل صفحات المصحف الشريف الـ 604.
* ترميز لوني تفاعلي يوضح حالة كل صفحة (حفظ ممتاز، حفظ جيد، يحتاج مراجعة، غير محفوظ).
* عرض تفصيلي لنسبة الإنجاز وعدد الصفحات المحفوظة وإجمالي الأجزاء.

### 2. المساعد القرآني الذكي (Gemini AI Chat) 🤖
* معلم وموجه تفاعلي يجيب على متشابهات الآيات ويوضح الفروق اللفظية.
* تقديم نصائح مخصصة للتغلب على صعوبات الحفظ والنسيان بأسلوب مشجع وإيماني.
* الاحتفاظ بسجل المحادثات في قاعدة البيانات المحلية.

### 3. مستشار التخطيط الشخصي (Onboarding Wizard) 🎯
* اختبار تفاعلي لتحديد نمط التعلم (سمعي، بصري، أو مختلط).
* بناء خطة حفظ مخصصة بناءً على مستوى الحفظ الحالي، الأهداف اليومية، والوقت المفضل للتنبيهات.

### 4. مساحة التسميع والمراجعة التفاعلية ✍️
* لوحة مخصصة لكل صفحة تتيح قراءة القرآن مع ميزات لتسجيل عدد الأخطاء ومقدار التردد أثناء التسميع.
* تقييم فوري وحساب تلقائي لدرجة جودة الحفظ (Memory Score).

### 5. مشغل التلاوات القرآني 🔊
* الاستماع لتلاوات الصفحة الحالية للمساعدة في التثبيت السمعي وتصحيح التجويد.

### 6. ملتقى الحفاظ (Community Feed) 👥
* مساحة لمشاركة المنشورات، الخواطر التفسيرية، والفوائد بين الحفاظ.
* إمكانية طرح الأسئلة والرد عليها لدعم الروح الجماعية والمنافسة الشريفة.

### 7. لوحة الإدارة والتحكم (Admin Panel) ⚙️
* مراقبة أداء الخادم وقاعدة البيانات.
* إدارة حسابات المستخدمين وإمكانية إعادة تهيئة البيانات أو أخذ نسخ احتياطية.

## 🛠️ التقنيات المستخدمة | Tech Stack

* **الواجهة الأمامية (Frontend):** React (Vite) + TS/JS + Tailwind CSS + Framer Motion (للحركات والانتقالات السلسة) + Lucide Icons.
* **الخلفية (Backend):** Express.js (Node.js) مع معالجة طلبات RESTful API وخدمات الأمان وتشفير كلمات المرور.
* **قاعدة البيانات (Database):** SQLite3 (قاعدة بيانات محلية سريعة وخفيفة لحفظ المستخدمين والتقدم والمحادثات والمنشورات).
* **الذكاء الاصطناعي (AI Integration):** Google Gemini Pro & Flash API عبر المكتبة الرسمية `@google/generative-ai`.

## 🚀 طريقة التشغيل | Installation & Setup

### 📋 المتطلبات الأساسية
* تثبيت بيئة عمل [Node.js](https://nodejs.org/).

### 🔧 الخطوات

1. **تثبيت الحزم البرمجية (Dependencies):**
   ```bash
   cd ma7fath-ai
   npm install
   ```

2. **إعداد ملف البيئة (`.env`):**
   قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع (أو قم بتعديل الملف الحالي) وأضف مفتاح Gemini API الخاص بك:
   ```env
   # مفتاح Google Gemini AI لتشغيل الذكاء الاصطناعي
   GEMINI_API_KEY=your_gemini_api_key_here

   # معرف عميل Google لـ OAuth (اختياري للتطوير المحلي)
   VITE_GOOGLE_CLIENT_ID=your_google_client_id_here
   ```

3. **تشغيل خادم الخلفية (Backend Express Server):**
   ```bash
   # من داخل مجلد ma7fath-ai
   node server/index.js
   ```
   *سيعمل خادم الخلفية على المنفذ `5000` (`http://localhost:5000`)*.

4. **تشغيل خادم التطوير للواجهة الأمامية (Frontend React):**
   افتح نافذة ترمينال جديدة ثم قم بتشغيل:
   ```bash
   npm run dev
   ```
   *ستعمل الواجهة الأمامية على المنفذ `5173` (`http://localhost:5173`)*.
